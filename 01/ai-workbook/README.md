# Module 01 - AI Workbook

1. **[problem.ipynb](./problem.ipynb)** - the explanation and the problem, with links to the problem program and cells that reproduce the bug and run your verification harness.

Program files:

- Problem program: [adversarial_vector_add_buggy.cu](./adversarial_vector_add_buggy.cu)

- Solution program(s):
  - [solutions/adversarial_vector_add_fixed.cu](./solutions/adversarial_vector_add_fixed.cu)
  - [solutions/verify_vector_add_solution.cu](./solutions/verify_vector_add_solution.cu)
